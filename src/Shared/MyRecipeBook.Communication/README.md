# MyRecipeBook.Communication
Camada compartilhada para contratos de comunicação.
- Contém DTOs, Requests e Responses usados entre API e Application.
- Não possui lógica de negócio.

# Função:

- Define contratos de comunicação (DTOs, Requests, Responses).
- Usada pela API e pela Application para padronizar entrada/saída.
- Não deve conter lógica de negócio.
